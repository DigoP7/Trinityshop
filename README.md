# Trinityshop
TrinityShop | E-commerce Full Stack 🛒  Sistema de gerenciamento de vendas desenvolvido com PHP e MySQL, focado em lógica de backend e modelagem de dados. A interface foi construída com HTML5 e CSS3 puro. Inclui fluxo de autenticação, níveis de acesso (User/ADM) e persistência de dados. Projeto focado em código limpo e funcional.
